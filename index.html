MSBA Living Portfolio

Synthesis of Coursework at Baruch College's Zicklin School of Business

Overview & Philosophy

This portfolio captures my academic journey through the MS in Business Analytics program at Baruch College’s Zicklin School of Business. It integrates lessons from three core courses—Database Management Systems (CIS 9340), Programming for Analytics (CIS 9650), and Managerial Statistics (STA 9708)—and showcases how these subjects interconnect to form a unified analytical mindset.

My goal is to demonstrate not just mastery of individual topics, but an appreciation for the ways that database design, programming skills, and statistical reasoning reinforce each other. By presenting conceptual maps, code snippets, diagrams, and reflections, I aim to create a professional and intellectually rich artifact that can be shared with peers, recruiters, and professors.

Course Summaries

CIS 9340 – Principles of Database Management Systems

The course explores the evolution of database management systems, entity‑relationship modelling, relational data models, SQL, transaction processing, concurrency control, system architectures, and legal/ethical issues [source].

CIS 9650 – Programming for Analytics

Focused on Python programming, this class covers data access, wrangling, exploration, basic visualization, and text mining [source].

STA 9708 – Managerial Statistics

The course introduces descriptive statistics, probability distributions, confidence interval estimation, hypothesis testing, and regression models [source].

Real‑World Applications in NYC

  • Finance & Fintech: SQL databases for transaction data, Python scripts for aggregation, statistical models for forecasting.
  • Healthcare Analytics: Secure patient databases, automated data pipelines, statistical tests for treatment efficacy.
  • Marketing & E‑Commerce: Customer data management, text mining of reviews, A/B testing.
  • Urban Tech & Smart Cities: Integrating sensor data, data cleaning, predictive analytics for planning.

Reflection & Next Steps

Together, these courses build a robust analytical toolkit. Learning to structure and query databases ensures data integrity; programming empowers automation and exploration; and statistics provide methods to quantify uncertainty and test hypotheses.

Moving forward, I plan to deepen my expertise by undertaking projects that integrate these skills — building data warehouses, developing dashboards, and exploring machine learning — and updating this portfolio accordingly.
